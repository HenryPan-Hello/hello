Setup guide

How to replace the sample games

This starter site is static, so replacing a game is easy.

1. Add your game files

Create a folder inside games/, for example games/my-new-game/. Put your exported HTML5 game files there.

games/
  my-new-game/
    index.html
    assets/
    scripts/

2. Update data/games.json

Add one object like this:

{
  "id": "my-new-game",
  "slug": "my-new-game",
  "title": "My New Game",
  "description": "Short game description.",
  "categories": ["Action", "Arcade"],
  "tags": ["fast", "boss", "platformer"],
  "coverEmoji": "⚔️",
  "featured": true,
  "gamePath": "games/my-new-game",
  "entryFile": "index.html"
}

3. Optional: remove the sample games

Delete the sample folders and remove their objects from data/games.json.

4. Deploy

You can upload this folder to GitHub Pages, Cloudflare Pages, Netlify, or any static host.

Notes

  • Best for HTML5 games.
  • Unity WebGL / Godot Web / Construct exports can work if all files stay together.
  • If your game needs a special start file, change entryFile.
